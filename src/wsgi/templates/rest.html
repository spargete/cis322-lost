petunia.cs.uoregon.edu:9090/rest/lost_key -- Returns the LOST public key for encryption and decryption purposes.
petunia.cs.uoregon.edu:9090/rest/activate_user -- Reactivates LOST access for a user or generates a new user account as needed.
petunia.cs.uoregon.edu:9090/rest/suspend_user -- Revokes access for a user. If the user does not exist or is already inactive, this call has no effect.
petunia.cs.uoregon.edu:9090/rest/list_products -- Requests a listing of all products in LOST based on a filter criteria. Additional filters may be provided and additional data may be returned. The requests of the minimum form documented in the example must be accepted and the produced result must have at least the structure and fields of example provided. Asset information may not be provided by this call (e.g. no returning number of assets that are of product types).
petunia.cs.uoregon.edu:9090/rest/add_product -- Adds products to LOST. Takes a JSON as input, returns either "OK" or "FAIL".
petunia.cs.uoregon.edu:9090/rest/add_asset -- Adds a new asset to LOST.
